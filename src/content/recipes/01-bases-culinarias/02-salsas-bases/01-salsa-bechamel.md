# Salsa Bechamel

## Ingredientes

| Cantidad | Unidad | Ingrediente     |
| -------- | ------ | --------------- |
| 0.050    | kg     | Mantequilla     |
| 0.050    | kg     | Harina          |
| 1        | l      | Leche           |
| 0.004    | kg     | Sal             |
| 0.001    | kg     | Pimienta blanca |
| 0.001    | kg     | Nuez moscada    |

### Opcional

| 0.002    | kg     | Tomillo     |

### Para el Croque Monsieur

| Cantidad | Unidad | Ingrediente   |
| -------- | ------ | ------------- |
| 0.500    | pza    | Pan baguette  |
| 0.150    | kg     | Jamón de pavo |
| 0.150    | kg     | Queso Gouda   |
| 0.015    | kg     | Mostaza Dijon |
| 0.025    | kg     | Perejil       |

## Pasos

1. Agregar a una olla la mantequilla, derretir y añadir la harina.
2. Elaborar un roux rubio o dorado.
3. Agregar poco a poco la leche.
4. Batir constantemente mientras se integra para eliminar los grumos.
5. Una vez que esté hirviendo, bajar a fuego lento.
6. Cocinar de 20 a 30 minutos o hasta conseguir la consistencia deseada.
7. Sazonar.

### Para el Croque Monsieur

1. Cortar en rebanadas el pan baguette.
2. Colocar en una charola.
3. Baguette + salsa bechamel + jamón + queso + pan baguette + salsa bechamel + queso.
4. Honrear hasta que el queso gratine y dore.
5. Decorar con perejil.
