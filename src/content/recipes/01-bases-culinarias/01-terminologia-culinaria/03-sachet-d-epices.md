# Sachet d'Épices (Saco de especias)

## Ingredientes

| Cantidad | Unidad | Ingrediente           |
| -------- | ------ | --------------------- |
| 0.500    | cm     | Manta de cielo        |
| 0.003    | kg     | Hojas de laurel       |
| 0.045    | kg     | Tomillo fresco        |
| 0.003    | kg     | Clavo entero          |
| 0.030    | kg     | Ajo                   |
| 0.100    | kg     | Perejil               |
| 0.002    | kg     | Pimienta negra entera |
| c/s      |        | Hilo algodón blanco   |

## Pasos

_Usar solo los tallos del perejil_

1. Cortar la manta de cielo en un cuadro regular.
2. Colocar las especias en el centro y cerrar la tela formando una bolsa.
3. Amarrar con el hilo para evitar que se dispersen en el líquido durante la cocción.

**NOTA: LA RECETA RINDE PARA TRES PIEZAS DE SACHETS**
